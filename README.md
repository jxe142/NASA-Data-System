System Design and Development

A site that makes NASA's Electric Arc Shock Tube (EAST) Test 59 Data# NASA-Data-System

Set Up
=============
1) Follow this link in order to set up your virualenv: https://packaging.python.org/guides/installing-using-pip-and-virtualenv/
2) Install requirements.text with:  pip install -r /path/to/requirements.txt
3) Set up your postgres database note you must isntall Postgres on your system first. Then follow this link https://www.digitalocean.com/community/tutorials/how-to-use-postgresql-with-your-django-application-on-ubuntu-14-04
      *Also note that the user we are using is the default psql user and the password should be password
